🔍 番号検索

📲 番号＋セラピスト名を送信 → 🔍 履歴を検索 → 📒 一覧で返信 → 📲 セラピストに転送
BEFOREシートを開いて Ctrl+F で検索→該当行を読む（3〜5分） → AFTERLINE送信で3秒

📝 日報更新

📲 テンプレ送信 → 📝 日報シート自動入力
BEFOREシートを開いて手打ち・行追加・書式調整（1件3分） → AFTERテンプレ送信で自動入力

FAQ

よくあるご質問

操作はむずかしくないですか？スタッフでも使えますか？
LINEを送るだけなので、新人スタッフでも初日から使えます。テンプレートをコピー＆貼り付けるだけで全機能が動きます。
料金体系・コース内容は店舗ごとに違うのですが対応できますか？
完全対応です。コース料金・オプション料金・指名料・歩合率――店舗ごとに細かく設定できます。
複数店舗を運営しているのですが、まとめて管理できますか？
むしろ複数店舗オーナーに最適です。月報更新コマンド一発で全店舗の売上・本数・出勤を瞬時に合算表示します。
セキュリティは大丈夫ですか？顧客情報が他店舗に漏れる心配は？
ご安心ください。LINE userId と契約管理番号（T-XXX）の二重認証を採用し、登録は必ずオーナーが手動承認します。認証されていないアカウントはBotに一切反応しません。また店舗ごとにアクセス権限が完全分離されており、A店舗のスタッフがB店舗のデータを見ることは構造的に不可能です。
トラブルが起きた時のサポートはどうなりますか？
LINEで直接ご連絡いただければ即対応します。機能追加・カスタマイズのご相談も同じ窓口でお受けしています。
解約はいつでも可能ですか？
いつでもご解約いただけます。長期契約の縛りはありません。
